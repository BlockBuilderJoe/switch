Connect to Cloudflare

Switch uses Cloudflare's free DNS service to filter websites. To set this up, we need a quick API token from your Cloudflare account.

  1. Click the button below to open Cloudflare
  2. Sign up for free (or log in)
  3. The token is pre-configured — just click "Create Token"
  4. Copy the token and paste it here
